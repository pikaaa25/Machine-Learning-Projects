# Fairness-Aware Disease Risk Prediction using Machine Learning

This project focuses on predicting disease risk using machine learning models while evaluating and improving fairness across sensitive attributes such as gender and age.

## Key Topics
- Disease risk prediction
- Fairness in machine learning
- Model evaluation and bias analysis

## Methods
- Logistic Regression
- Random Forest
- Fairness-aware evaluation metrics

## Results
The models demonstrate competitive predictive performance while highlighting disparities across demographic groups, motivating fairness-aware analysis.

## Conclusion
This project emphasizes the importance of fairness and transparency when deploying machine learning models in healthcare applications.
